## Regras de Negócio
1. Não é permitido cadastrar clientes com o nome vazio.
2. Não é permitido cadastrar horários com valores negativos.
3. Não é possível reservar um horário que já esteja ocupado por outro cliente.
4. O sistema deve calcular automaticamente o valor total do aluguel (somando os valores dos horários escolhidos pelo cliente no dia).
5. O sistema deve permitir a consulta e listagem de todos os aluguéis feitos em um determinado dia.
